Afin d'éxécuter nos tests, nous avons suivi les étapes suivantes sur Visual Studio Code.

        - Ouvrir spécifiquement le dossier https://github.com/mildshield14/IFT3913/tree/main/ift3913_tp4%20remise/currency-converter-in-java-master 
          Ceci est une étape cruciale car Visual Studio Code ne détecte pas les tests avec un repo cloné de GitHub, donc avec notre rapport+README
          
        - Chercher le symbole d'une flaque de chimiste. Si vous avez déjà les extensions requises, VS Code va vous suggérer des options. Choisissez Java.
          Sinon, cliquez sur "Install Additional Test Extensions...". Téléchargez Test Runner for Java. Autre option, téléchargez sur 
          https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-test.

        - Vous verrez une option avec le symbole Run en passant votre cursor sur currency-converter-in-java-master. Cliquez et patientez pendant l'éxécution des tests.
        - Spoiler : nos tests échouent, plus de détails disponible dans le rapport

A noter, qu'une image de l'éxécution de nos tests est disponible dans le rapport.
